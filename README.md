A lightning-fast, asynchronous IP scanner designed to detect CDN (Content Delivery Network) IPs with zero limitations. Built with pure Python - no ping dependencies, no external binaries, works everywhere.

---

## ✨ Features

- 🚀 **Blazing Fast** - 500 concurrent threads by default, async I/O powered by `aiohttp`
- 🎯 **CDN Detection** - Identifies Cloudflare, CloudFront, Akamai, and Fastly
- 🌐 **No Ping Required** - Pure HTTP/HTTPS checking, zero subprocess issues
- 🖥️ **Cross-Platform** - Works on Windows, Linux, and macOS flawlessly
- 📊 **Real-time Progress** - Live results with color-coded output
- 🛑 **Ctrl+C Safe** - Graceful interruption with automatic result saving
- ♾️ **No Limits** - Scan as many IPs as you want
- 📝 **Multiple Input Formats** - CIDR, ranges, and single IPs supported
